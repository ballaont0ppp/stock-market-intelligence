# Product Overview

## Stock Market Prediction & Portfolio Management Platform

A comprehensive web application that combines machine learning-based stock price prediction with full-featured portfolio management capabilities.

## Core Features

**Prediction Engine**
- Stock price forecasting for next 7 days using ARIMA, LSTM, and Linear Regression models
- Sentiment analysis integration from social media (Twitter)
- Support for NASDAQ and NSE stocks

**Portfolio Management**
- Real-time portfolio tracking and performance analytics
- Buy/sell order execution with commission tracking
- Wallet management (deposits, withdrawals)
- Holdings management with unrealized gains/losses
- Transaction history and audit logs

**User Management**
- Role-based access (Admin, User)
- User profiles with risk tolerance and investment goals
- Notification system for portfolio events
- Dividend tracking and automatic processing

**Analytics & Reporting**
- Performance metrics (returns, win rate, annualized returns)
- Sector allocation visualization
- Transaction reports and billing statements
- Real-time stock price monitoring

## User Roles

**Users**: Portfolio management, stock predictions, real-time prices, news, currency converter
**Admins**: All user features plus user management, company/broker management, dividend management, system monitoring, audit logs
